시작 상태

핀 프로젝트가 3개 이하라면 3개가 될 때까지 복제한다.

while(projectCount < 3)
0번 아이템을 복제해서 push 한다.
1번 아이템을 복제해서 push 한다.

for(let i=0; arr.length < 3; ++i) {
    arr.push(arr[i]);
}

처리 이후 -1(맨 마지막 아이템)을 맨 앞에 push한다. arr.unshift(arr.pop())

arr.unshift(arr.pop());


그렇게 [ -1, 0, 1] 상태로 시작한다.

:렌더링

-1 : 왼쪽에 표시한다. 블러처리 된다.

0 : 중앙에 표시한다.

1 : 오른쪽에 표시한다. 블러처리 된다.
2부터는 표시하지 않는다.

:이후

왼쪽 버튼을 누를 시 arr.unshift(arr.pop()) unshift:인자로 들어온 값을 맨 앞에 집어넣는다.

오른쪽 버튼 누를 시 arr.push(arr.shift()) shift:맨 처음 엘리먼트를 뽑아내서 제거하여 리턴한다.
